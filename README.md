## 🧠 Overview  

**IntelliMesh AI** is an **agentic decision system** built for enterprise-grade AI architectures.  

It combines:  
- 🧠 Decision Intelligence (DecisionCore)  
- 🤖 Multi-Agent Orchestration  
- 📄 Document Intelligence Pipelines  

→ into a structured execution system:  

Planner → Researcher → Analyst → Writer → Reviewer


- Role-based execution  
- Pipeline-driven task flow  
- Modular agent design  

---

### 🧠 Decision Intelligence (DecisionCore)  

- Policy-driven decision layer  
- Strategy + execution alignment  
- Structured reasoning frameworks  

---

### 📄 Document Intelligence  

- PDF ingestion (PyMuPDF)  
- Keyword extraction  
- Context-aware processing  
- Multi-document handling  

---

### 📊 Evaluation & Metrics  

- Complexity scoring  
- Clarity & coverage metrics  
- Confidence scoring  
- Structured output evaluation  

---

### 📈 Execution Dashboard  

- Multi-agent output visualization  
- Session-based tracking  
- Structured insights layer

## 🔄 System Workflow  

Input (Text / PDF)
         ↓
Context Processing + Keyword Extraction
         ↓
Decision Layer (Policy + Strategy)
         ↓
Agent Layer (Planner → Researcher → Analyst → Writer → Reviewer)
         ↓
Evaluation Layer (Metrics + Scoring)
         ↓
Dashboard + Export (JSON)
